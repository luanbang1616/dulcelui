# Requerimientos - Dulce Lui

## 1. Información General
- **Nombre de la tienda**: Dulce Lui
- **Tipo de negocio**: Pastelería y repostería artesanal
- **Productos**: Cupcakes, brownies, tortas, cheesecakes, maní surtido mixto
- **Servicios**: Pedidos por encargo
- **Estilo visual**: Clásico y elegante

## 2. Colores
- **Icy Blue**: `#B3E6GB` ( Azul hielo )
- **Berry Red**: `#C21121` ( Rojo berries )

## 3. Secciones Requeridas

### 3.1 Inicio (Hero Section)
- Título principal con el nombre "Dulce Lui"
- Subtítulo atractivo
- Imagen principal / banner
- Animación fluida de entrada

### 3.2 Misión / Visión / Objetivos
- **Misión**: Descripción de la misión de la tienda
- **Visión**: Visión a futuro
- **Objetivos**: Lista de objetivos principales

### 3.3 Productos
- **Cupcakes**: Variados sabores y decoraciones
- **Brownies**: Clásicos y con variaciones
- **Tortas**: Para ocasiones especiales
- **Cheesecakes**: Diferentes sabores
- **Maní surtido mixto**: Variedad de opciones
- Cada producto debe incluir imagen

### 3.4 Contacto
- **Horario**: Lunes a viernes de 8 AM a 3 PM
- **Correo electrónico**: Información de contacto
- **WhatsApp**: Número para pedidos
- **Botón "Realizar pedido"**: Link directo a WhatsApp
- **Mapa de ubicación**: Ubicación de la tienda

## 4. Requisitos Técnicos

### 4.1 Framework
- React con Vite

### 4.2 Estilos
- Diseño responsivo (mobile, tablet, desktop)
- Glassmorphism (efecto de vidrio)
- Animaciones fluidas (transiciones suaves)
- Footer elegante

### 4.3 Imágenes
- Imágenes de productos (placeholder desde fuentes públicas)
- Logo de la tienda (diseño sencillo y clásico)

## 5. Animaciones
- Transiciones suaves entre secciones
- Efectos hover en botones y cards
- Animación de entrada para elementos
- Smooth scroll entre secciones

## 6. Responsividad
- Mobile: < 768px
- Tablet: 768px - 1024px
- Desktop: > 1024px
